Autonomous Logic

No remote needed. The ESP8266 analyzes sensor data every 3 seconds to make independent movement decisions.

DECISION LOOP

01. SCAN ENVIRONMENT

Servos sweep X and Y axes. The system records voltage values at 5-degree increments.

02. COMPUTE MAXIMA

The algorithm identifies the coordinates (Angle X, Angle Y) that yielded the highest voltage.

03. EXECUTE MOVEMENT

Panels align. If efficiency is still low, the wheels turn to chase the light source.
